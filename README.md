# React de Cero a Experto (React 19 + TypeScript)

Repositorio personal con los ejercicios, retos y proyectos del curso  
**"React: de Cero a Experto (Hooks, TypeScript, MERN, Testing, AI, etc.)"**  
impartido por **Fernando Herrera** ({d/t} DevTalles).

El objetivo de este repo es ir guardando el código limpio, organizado y versionado a medida que avanzo por las secciones del curso.

---

## Estructura del repositorio

> Los nombres de carpetas son orientativos, puedes adaptarlos según cómo organices tu código.

- `01-reforzamiento/`  
  Repaso de JavaScript, TypeScript y conceptos básicos necesarios para React.

- `02-first-steps/`

## Tecnologías y temas trabajados

Algunos de los temas y herramientas que se practican en los ejercicios:

- **React 19+** y **TypeScript**
- Hooks básicos y avanzados + **custom hooks**
- Functional Components, HOC (higher-order components)
- **Context API**, **Redux Toolkit**, **Zustand**
- **MERN**: MongoDB + Express + React + Node
- **React Router v7+**, múltiples routers y layouts
- UIs con **TailwindCSS**, **ShadCN**, **MUI**, Bootstrap
- Animaciones en componentes
- Integraciones con AI para generación de interfaces
- **JWT**, encriptación de contraseñas, Google Sign-in, autenticación personalizada
- Pruebas con **Vitest** + **React Testing Library**
- Mocks, spies, snapshots y pruebas especializadas por ejercicio
- CRUDs completos, logs, buenas prácticas y despliegues a producción
- Uso de herramientas como React DevTools, Redux DevTools, Git y GitHub
