# Heroes APP

This is a simple React application that displays a list of heroes and their details. The app allows users to view hero information, including their name, power, and biography.

## Launching the App
To launch the app, follow these steps:
1. Clone the repository to your local machine:
   ```bash
   git clone
    ```
2. Navigate to the project directory:
   ```bash
    cd 05-heroes-app
    ```
3. Edit Enviroment Variables:
     - Create a `.env` file in the root directory of the project.
     - Add any necessary environment variables as needed for your setup.
4. Install the dependencies:
    ```bash
    npm install
    ```
5. Start the development server:
   ```bash
    npm run dev
    ```
